## Use in this repository

This repository (`Eiasash/Geriatrics`, public PWA at https://eiasash.github.io/Geriatrics/) includes a subset of GRS8 Book 3 (Q&A) questions, filtered to topics aligned with the IMA Geriatrics syllabus v3.0 (July 2024) coverage gaps:

- **ti=43**: Andropause / sexuality and the aging man (Hazzard Ch 36–37)
- **ti=44**: Prevention and health promotion (Hazzard Ch 10)
- **ti=45**: Geriatric team / interdisciplinary care (Hazzard Ch 11, 17, 19)

Each GRS8-derived question carries:
- `t: "GRS8"` tag
- `ref: "GRS8 Q#NNN"` reference to the original question number
- Inline attribution at the end of the explanation field (`e`)
